# Liveklass Extensions — 정책 문서

> 이 문서는 확장 기능 마켓의 목적, 대상, 운영 원칙을 정의합니다.
> 페이지 카피, UX 구조, 기능 결정의 기준 문서입니다.

---

## 1. 목적

라이브클래스에 공식적으로 없는 기능을 **오픈소스 확장**으로 제공하여,
고객사가 직접 적용하거나 필요 시 라이브클래스 지원을 받을 수 있게 한다.

**핵심 가치:** 고객이 기능 부재로 이탈하지 않도록, 검증된 우회 방법을 체계적으로 공개한다.

**페이지 성격:** 세일즈 도구 — 담당자가 고객에게 URL을 직접 공유하는 방식으로 유통.
공홈/메인페이지에서 링크하지 않으며, URL을 아는 사람은 누구나 접근 가능.
향후 로그인 게이팅으로 접근 제한 가능.

---

## 2. 대상

- **1차 대상:** 라이브클래스 기존 주요 고객사 (운영 중인 사이트 보유)
- **2차 대상:** 신규 고객사 (도입 검토 중, 기능 부재가 허들인 경우)
- **공통 전제:** 라이브클래스 사이트를 운영 중이거나 운영 예정인 사업자
- **유통:** 세일즈 담당자가 상담/컨설팅 과정에서 URL 직접 공유

---

## 3. 가격 정책

- **확장 기능 자체:** 무료 (오픈소스)
- **셀프 적용:** 가이드 문서 제공, 비용 없음
- **라이브클래스 지원 적용:** 무료 — 고객 성공 차원에서 제공
  - 고객사가 직접 하기 어려운 부분만 라이브클래스에서 셋업
  - 향후 로그인 게이팅으로 대상 자체를 제한하여 무료 유지

---

## 4. 확장 기능 품질 기준

- **모든 확장 기능은 베타** — 공식 기능이 아님을 명시
- 각 기능에 대한 **피드백을 상시 수집하고 반영** (마켓 내 기능별 피드백 폼)
- 기능별 명시 항목:
  - 적용 조건 (필요 권한, 외부 서비스 연동 등)
  - 알려진 제한 사항
  - 피드백 채널 (마켓 페이지 내 폼)

---

## 5. 마켓 노출 기준

> 어떤 기능이 마켓에 올라갈 수 있는가?

- [x] 최소 1곳 이상 테스트 또는 운영 적용 완료
- [x] 적용 가이드 문서 존재
- [x] 제한 사항 명시 완료
- **비공개 기능:** 마켓에 노출하지 않음. 담당 세일즈가 직접 안내

## 6. 고객 여정

```
발견 → 이해 → 적용 시도 → (막히면) 지원 요청 → 적용 완료 → 피드백
```

| 단계 | 고객 행동 | 마켓이 제공해야 하는 것 |
|------|----------|----------------------|
| 발견 | 세일즈에게 URL 받고 방문 | 기능 목록 + "무료 베타" 안내 + 내 상황에 맞는지 빠른 판단 |
| 이해 | 상세 확인 | 뭘 하는 건지, 뭐가 필요한지, 뭐가 안 되는지 |
| 적용 시도 | 가이드 보고 직접 적용 | 단계별 적용 가이드 (셀프서브) — **1차 CTA** |
| 지원 요청 | 직접 하기 어려운 부분 요청 | 지원 요청 폼 — **2차 CTA** |
| 적용 완료 | 사이트에 기능 동작 | 적용 확인 체크리스트 |
| 피드백 | 개선 의견 전달 | 기능별 피드백 폼 (마켓 페이지 내) |

---

## 7. 성공 지표

| 지표 | 측정 방법 |
|------|----------|
| 요청 수 | Supabase requests 테이블 row count |
| 실제 적용 수 | requests.status = '완료' count |
| 전환율 | 적용 완료 / 요청 수 |
| 셀프서브 비율 | 가이드만 보고 적용한 건수 (측정 방법 TBD) |
| 피드백 수 | 기능별 피드백 접수 건수 |

---

## 8. 페이지 재설계 방향

| 현재 | 문제 | 수정 방향 |
|------|------|----------|
| "내부 팀이 셋업합니다" | managed service 톤 | "가이드 보고 직접 적용 + 어려우면 무료 지원" |
| "설치 요청" CTA | 지원이 1차 행동처럼 보임 | 1차: "적용 가이드 보기", 2차: "지원 요청" |
| 포스터 "마켓 노출 기준" 카드 | 내부 운영 정보, 고객 무관 | 제거 → "무료 베타" 안내로 전환 |
| 가격 정보 없음 | "돈 드는 건가?" 불안 | "모든 확장 기능은 무료입니다" 명시 |
| 피드백 채널 없음 | 베타인데 의견 보낼 곳 없음 | 기능별 피드백 폼 추가 |
| 적용 가이드 없음 | 셀프서브가 기본인데 가이드 없음 | 기능별 적용 가이드 링크 필수 |
| 폼이 유일한 액션 | 셀프서브 경로 없음 | 가이드 → 셀프 적용 경로 추가 |
